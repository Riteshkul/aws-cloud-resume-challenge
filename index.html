Explainable Image Caption Generator

In our project, we've split it into two key parts: the Generation Module and the Explanation Module.**Generation Module**: This part focuses on creating image captions. It utilizes an encoder ( VGG16) to extract image features and a decoder ( LSTM) to generate text. The encoder processes the image, and the decoder turns those features into a descriptive caption.**Explanation Module**: This is where our innovation comes in. We introduce the "Interpretability Enhancement Model." This model enhances the interpretability of the generated captions. It takes inputs from both the Object Detection component (detecting regions of interest in the image) and the Decoder (words generated so far). Using these inputs, it generates a meaningful caption, providing insight into how it arrived at the description based on detected objects and generated content. This module aims to create captions that are not only informative but also easier to understand, bridging the gap between the image and the generated text.

Key Modules:- Generation Module Explaination Module Interpretability Enhancement Module. Object Detection Module
